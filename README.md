# my-portfolio

This is the code of my personal website made with Vue, threejs, gsap and a lot more.
I hope you enjoy it.

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
